## 🧠 Concepts Learned
- Machine Learning is about training models to learn patterns from data
- Supervised learning uses labeled data (input + output)
- Unsupervised learning works with unlabeled data to find hidden patterns
- Regression is used for continuous output (e.g., price prediction)
- Classification is used for categorical output (e.g., spam or not spam)
- Clustering groups similar data points together
- Deep Learning is a subset of ML that uses neural networks
